# goit-js-hw-06

OOP, classes, prototypes

## Completed Tasks

### Task 1: Customer Object

Implementation of a customer object with methods for managing balance,
discounts, and orders.

- `getBalance()` - Returns current balance
- `getDiscount()` - Returns current discount percentage
- `setDiscount(value)` - Sets new discount percentage
- `getOrders()` - Returns array of orders
- `addOrder(cost, order)` - Adds order with discount applied

### Task 2: Storage Class

Implementation of a Storage class with private items array.

- `constructor(item)` - Initialize storage with items
- `getItems()` - Returns all items
- `addItem(newItem)` - Adds new item to storage
- `removeItem(itemToRemove)` - Removes item from storage

### Task 3: StringBuilder Class

Implementation of a StringBuilder class for string manipulation with private
value.

- `constructor(initialValue)` - Initialize with string
- `getValue()` - Returns current string value
- `padStart(str)` - Prepends string to value
- `padEnd(str)` - Appends string to value
- `padBoth(str)` - Appends and prepends string to value
